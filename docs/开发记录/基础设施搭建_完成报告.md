## 项目结构

```
src/
├── main/                  # 主进程
│   ├── database/          # 数据库相关
│   │   └── index.ts       # 数据库初始化和管理
│   ├── ipc/               # IPC 通信处理
│   │   └── handlers.ts    # IPC 处理函数
│   └── index.ts           # 主进程入口
├── preload/               # 预加载脚本
│   └── index.ts           # 预加载脚本入口
└── renderer/              # 渲染进程
    ├── components/        # 通用组件
    │   ├── Layout.tsx     # 布局组件
    │   ├── Sidebar.tsx    # 侧边栏组件
    │   └── ToolContainer.tsx # 工具容器组件
    ├── modules/           # 工具模块
    ├── store/             # 状态管理
    │   └── index.ts       # Zustand store
    ├── styles/            # 样式文件
    ├── App.tsx            # 应用入口
    └── main.tsx           # 渲染进程入口
```
