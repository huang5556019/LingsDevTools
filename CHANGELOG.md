# Changelog

## [0.1.0] - 2026-04-25

### Added
- 编码解码工具：Base64 编码/解码、URL 编码/解码、JSON 格式化/压缩
- 数据处理工具：时间戳转换、正则表达式测试
- 网络工具：HTTP 请求测试、WebSocket 调试
- 历史记录管理：查看工具使用历史、管理收藏的工具配置
- 统一的错误处理机制
- 优化的用户界面和交互体验
- 性能优化：应用启动速度、工具操作响应时间
- 跨平台支持：Windows 和 macOS

### Changed
- 优化了应用启动逻辑，提高启动速度
- 统一了工具布局和样式
- 增加了加载状态指示器
- 优化了按钮和表单元素的样式和交互体验
- 增加了操作反馈动画
- 统一了成功提示样式
- 优化了工具状态保存机制

### Fixed
- 修复了编码解码工具的错误处理
- 修复了数据处理工具的错误处理
- 修复了网络工具的错误处理
- 修复了历史记录管理的错误处理
- 修复了应用在各种错误情况下的崩溃问题

### Security
- 增强了应用的错误处理和异常捕获机制
- 优化了网络请求的安全性
