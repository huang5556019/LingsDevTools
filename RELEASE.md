# DevTools 发布说明

## 版本 0.1.0

### 发布日期
2026-04-25

### 发布内容

DevTools 是一个跨平台开发常用工具客户端，提供了以下功能：

1. **编码解码工具**
   - Base64 编码/解码
   - URL 编码/解码
   - JSON 格式化/压缩

2. **数据处理工具**
   - 时间戳转换
   - 正则表达式测试

3. **网络工具**
   - HTTP 请求测试
   - WebSocket 调试

4. **历史记录管理**
   - 查看工具使用历史
   - 管理收藏的工具配置

### 系统要求

- **Windows**: Windows 10 及以上
- **macOS**: macOS 10.15 及以上

### 安装说明

1. **Windows**:
   - 下载 `DevTools-0.1.0-win-x64.exe` 安装包
   - 双击运行安装程序
   - 按照安装向导完成安装

2. **macOS**:
   - 下载 `DevTools-0.1.0-mac-x64.dmg` 或 `DevTools-0.1.0-mac-arm64.dmg` 安装包
   - 双击打开 DMG 文件
   - 将 DevTools 拖放到 Applications 文件夹

### 已知问题

- 部分网络环境下 WebSocket 连接可能不稳定
- 大文件处理时可能会占用较多内存
